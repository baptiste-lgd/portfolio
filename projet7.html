🖧 Déploiement d’une infrastructure Active Directory sous Windows Server 2022

Projet BTS SIO – Administration système et réseaux

Ce projet consiste à déployer un contrôleur de domaine sous Windows Server 2022 en mode Server Core, entièrement administré en ligne de commande avec PowerShell, au sein d’un environnement virtualisé. Il permet de comprendre la mise en place d’une infrastructure Active Directory, la gestion réseau ainsi que l’administration centralisée des utilisateurs et des postes clients.

📦 Mise en place de la VM

  • Création de la machine virtuelle sous Proxmox VE
  • Allocation des ressources (RAM, CPU, stockage)
  • Montage de l’image ISO Windows Server 2022
  • Préparation de l’environnement d’installation

⚙️ Installation du système

  • Installation de Windows Server 2022 en mode Server Core
  • Configuration initiale via SConfig
  • Définition du nom du serveur
  • Paramétrage du système exclusivement en ligne de commande

🌐 Configuration réseau

  • Attribution d’une adresse IP statique via PowerShell
  • Configuration de la passerelle et du DNS
  • Vérification de la connectivité réseau
  • Mise en place de l’accès distant (SSH)

🏢 Déploiement Active Directory

  • Installation du rôle AD DS en PowerShell
  • Création du domaine "legrand.lan"
  • Promotion du serveur en contrôleur de domaine
  • Configuration automatique du service DNS

🧱 Organisation et gestion

  • Création des unités d’organisation (OU) en PowerShell
  • Structuration des utilisateurs et des ordinateurs
  • Intégration d’un poste client au domaine
  • Gestion et déplacement des objets Active Directory

🔐 Administration et sécurisation

  • Installation et configuration du service SSH
  • Authentification par clé sécurisée
  • Gestion des permissions système
  • Supervision des services critiques via PowerShell

Ce projet a été entièrement réalisé en ligne de commande avec PowerShell, sans interface graphique. Il m’a permis de développer une approche plus avancée de l’administration Windows Server, notamment dans le déploiement automatisé d’un contrôleur de domaine et la gestion d’une infrastructure Active Directory.
