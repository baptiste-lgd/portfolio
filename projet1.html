🚀 Configuration Git sur Debian VM

1️⃣ Se placer dans le répertoire de travail

Ouvrez un terminal et naviguez vers votre répertoire :

cd /home/proxmox

2️⃣ Installer Git

Vérifiez ou installez Git :

sudo apt update
sudo apt install git -y

3️⃣ Configurer Git

Définissez votre identité pour les commits :

git config --global user.name "Votre Nom Prénom"
git config --global user.email "votre.email@example.com"
Le --global applique la config à tous vos projets.

4️⃣ Ajouter votre clé SSH

Authentifiez vos opérations Git :

ssh-add ~/.ssh/proxmox
Assurez-vous que la clé existe et est autorisée sur GitHub.

5️⃣ Cloner le dépôt GitHub

Clonez le dépôt :

git clone git@github.com:apibim-xyz/web-02.git

Un dossier web-02 sera créé avec tout le projet.

6️⃣ Vérification

Vérifiez la configuration :

git config --list

Vérifiez que le dépôt a bien été cloné :

ls web-02
