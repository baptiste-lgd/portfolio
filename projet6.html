🐙 Git & GitHub sur machine virtuelle Debian

Projet BTS SIO – Gestion de version et accès SSH

Ce projet consiste à mettre en place et utiliser Git sur une machine virtuelle Debian afin de gérer un dépôt GitHub en SSH. Il permet de comprendre le fonctionnement du contrôle de version et de l’authentification sécurisée via clé SSH.

📁 Préparation de l’environnement

  • Accès à la machine virtuelle Debian
  • Choix du répertoire de travail
  • Organisation des dossiers de projet

⚙️ Installation et configuration de Git

  • Installation de Git via apt
  • Vérification de l’installation
  • Configuration du nom d’utilisateur
  • Configuration de l’adresse email

🔐 Authentification SSH

  • Ajout de la clé SSH dans l’agent
  • Vérification de la clé ~/.ssh/proxmox
  • Connexion sécurisée à GitHub

📦 Clonage et utilisation du dépôt

  • Clonage du dépôt GitHub en SSH
  • Création automatique du dossier projet
  • Vérification des fichiers clonés
  • Validation de la configuration Git

Ce projet m’a permis de comprendre l’utilisation de Git en environnement Linux ainsi que la gestion sécurisée de dépôts GitHub via SSH.
